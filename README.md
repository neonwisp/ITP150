# ITP150 Python Programming

## Overview

This repository contains coursework, notes, programming exercises, and projects completed for ITP150. The purpose of this repository is to document my progress in learning Python programming and fundamental software development concepts.

## Repository Contents

### Class Notes
Course notes, study materials, and reference documents.

### Labs
Hands-on assignments completed throughout the course.

### Python Programs
Practice programs covering topics such as:

- Variables and data types
- Input and output
- Arithmetic operations
- Conditional statements
- Loops
- Functions
- Problem-solving techniques

## Purpose

This repository serves as:

- A record of coursework and assignments
- A collection of programming exercises
- A study resource for reviewing Python concepts
- A demonstration of growth and progress throughout the course
